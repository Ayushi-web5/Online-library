# Online Library React Application

## Project Description
This project is a simple Online Library System built using React and Vite.  
Users can browse books, search for books, view detailed information, and add new books to the library.

## Features
- Home page with popular books
- Browse books with search functionality
- Dynamic routing for book details
- Add new books using Redux state management
- 404 page for invalid routes
- Responsive card layout styling

## Technologies Used
- React
- Vite
- Redux Toolkit
- React Router
- CSS
